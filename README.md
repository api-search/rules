# API Search - Rules

This repository stores Spectral API governance rules for the [APIs.io](https://apis.io) search network.

## About

Rules define API governance and linting standards using Spectral rulesets. They are generated from api-evangelist repos and used to build the `_rules/` Jekyll collection on the APIs.io network site.

## Related Repositories

- [apis](https://github.com/api-search/apis) - Individual API records
- [schemas](https://github.com/api-search/schemas) - JSON Schema references
- [providers](https://github.com/api-search/providers) - API provider records
- [network](https://github.com/api-search/network) - The APIs.io network site
